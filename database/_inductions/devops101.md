# What is devops?

* Developer Operations
* Monitorointi
* Automaatio
* Prosessikehitystä
    * Prosessin valvominen
* Hallinnollisten yksikköjen yhteistyön vahvistaja
* Dokumentointi
* Tuotekehityksen palvelemista
* Testaus
* Poikkifunktionaalista työskentelyä
